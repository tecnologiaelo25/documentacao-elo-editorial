Solicitação de Exclusão de Dados de Usuário

Esta página fornece as instruções para que você, usuário do assistente virtual "Professora Eloísa" do Grupo Elo Editorial no WhatsApp, possa solicitar a exclusão completa de seus dados de nossos sistemas, em conformidade com a Lei Geral de Proteção de Dados (LGPD) e as políticas da plataforma Meta.

Como solicitar a exclusão dos seus dados:

Para iniciar o processo de exclusão, por favor, siga os passos abaixo:

  1. Envie um e-mail para o nosso Encarregado de Proteção de Dados no endereço:
    danilo.molero@eloeditora.com.br
  2. No campo "Assunto" do e-mail, por favor, utilize o seguinte título:
    Solicitação de Exclusão de Dados do WhatsApp
  3. No corpo do e-mail, é essencial que você informe o número de telefone completo (com código de país + DDD) que você utilizou para interagir com o nosso assistente. Este dado é necessário para localizarmos seu histórico de conversas e garantir a remoção correta.

O que acontece depois?

Após o recebimento do seu e-mail, nossa equipe processará sua solicitação. Seus dados serão permanentemente removidos de nossos bancos de dados em um prazo de até 15 dias úteis. Você receberá uma notificação por e-mail confirmando que a exclusão foi concluída.
